== Description ==

Plug in to add a contribution box of twitter to last of the online posting text.

== Installation ==

1. Upload the entire `post-retweet` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Post ReTweet' menu in your WordPress admin panel.
